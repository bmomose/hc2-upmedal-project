# Desafio 3 / Hirincoders 2 (parte 2) - Projeto UPMedal

Projeto desenvolvido para o desafio "E-commerce para o cliente UPMedal.com" do programa Hiring Coders #2 (2021), com parceria da Gama Academy e VTEX.

## Aplicação

A aplicação é uma réplica da loja de artigos esportivos baseado no próprio layout da loja real da UPMedal, utilizando ferramentas do VTEX IO para a sua criação.
